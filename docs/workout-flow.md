# Бизнес-флоу тренировок

## Создание шаблона
1. Пользователь создаёт шаблон.
2. Добавляет упражнения.
3. Добавляет плановые подходы (reps/time).
4. Сохраняет шаблон.

## Старт тренировки из шаблона
1. Создаётся WorkoutSession.
2. TemplateExercise → WorkoutExercise.
3. TemplateSet → SetEntry (weight = null).
4. Пользователь заполняет фактические данные.

## Редактирование активной тренировки
Допускается:
- добавление/удаление упражнения,
- добавление/удаление подхода.

Не допускается:
- изменение шаблона через тренировку.

## Прогресс
Для упражнения рассчитывается:
- максимальный вес/время в **последней тренировке**.
