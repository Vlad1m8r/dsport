# Доменная модель

## Основная идея
Система оперирует двумя ветками данных:
- План тренировок (Templates)
- Фактические тренировки (Workout Sessions)

## Сущности

### Exercise
Справочник упражнений:
- системные (ownerUserId = null)
- пользовательские (ownerUserId = userId)

Упражнение может относиться к нескольким группам мышц.

### WorkoutTemplate (План)
Шаблон тренировки:
- содержит упражнения,
- содержит плановые подходы,
- не хранит веса.

### WorkoutSession (Факт)
Конкретная тренировка пользователя:
- может быть создана из шаблона,
- может быть пустой,
- хранит фактические данные.

## Ключевой принцип
Template → копируется → WorkoutSession  
Template **никогда не изменяется** при выполнении тренировки.
